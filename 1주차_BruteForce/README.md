# 😝브루트포스(brute forc)

brute: 무식한, force: 힘 무식하게 힘으로 푼다.

완전탐색 알고리즘 으로 모든 경우의 수를 모두 탐색하면서 요구 조건에 충족되는 결과만을 가져온다.

## ❗방법

1. 순차 탐색
2. DFS
3. BFS
